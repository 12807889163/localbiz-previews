📧 Google Apps Script Setup

Step-by-step guide to enable form email submissions for Orange Agency

1Open Google Apps Script

Go to script.google.com and sign in with your Google account.

Click the + New project button.

2Paste the Apps Script Code

Delete any code already in the editor. Then copy the code below and paste it in:

// ============================================================ // Orange Agency — Info Form Email Receiver // Paste this entire code into script.google.com // ============================================================ const RECIPIENT_EMAIL = 'john@orangeagency.media'; const SENDER_NAME = 'Orange Agency Form'; // Called when the web app receives a POST request function doPost(e) { try { const payload = JSON.parse(e.postData.contents); const subject = payload.business_name ? `🌐 New Website Info — ${payload.business_name}` : '🌐 New Website Info Submission'; let body = '=== ORANGE AGENCY — WEBSITE INFO FORM ===\n\n'; const fields = [ ['Business Name', payload.business_name], ['Contact Name', payload.contact_name], ['Phone', payload.phone], ['Email', payload.email], ['Address / Service Area', payload.address], ['Preferred Domain', payload.domain], ['Google Business Profile', payload.google_business], ['Testimonials', payload.testimonials], ['Extra Requests', payload.extra], ['Source URL', payload.source], ['Submitted At', payload.submittedAt], ]; fields.forEach(([label, value]) => { if (value) { body += `${label}: ${value}\n`; } }); body += '\n--- End of Submission ---\n'; MailApp.sendEmail(RECIPIENT_EMAIL, subject, body, { name: SENDER_NAME, noReply: true, }); return ContentService .createTextOutput(JSON.stringify({ status: 'success' })) .setMimeType(ContentService.MimeType.JSON); } catch(error) { return ContentService .createTextOutput(JSON.stringify({ status: 'error', message: error.message })) .setMimeType(ContentService.MimeType.JSON); } } // Test function — run this to verify email sending works function testEmail() { MailApp.sendEmail( RECIPIENT_EMAIL, '✅ Test Email from Apps Script', 'Google Apps Script is configured correctly!', { name: SENDER_NAME } ); Logger.log('Test email sent.'); }

Tip: Click "+ New project" first, then paste. The default myFunction can be deleted.

3Save and Name the Project

Click the Untitled project text at the top left, rename it to something like Orange Agency Form, and click Rename.

4Deploy as Web App

At the top right, click Deploy ▼ → New deployment.

In the deployment panel:

  • Click the Select type gear icon → choose Web app
  • Description: "Info Form Receiver"
  • Execute as: Me (your Google account)
  • Who has access: Anyone
⚠️ Important: "Anyone" means anyone with the URL can submit the form. This is correct — the form is public-facing. The script only sends email; it doesn't expose your Gmail credentials.

Click Deploy. Google will ask you to authorize permissions — click Authorize and grant access (it needs to send email on your behalf).

5Copy the Web App URL

After deploying, you'll see a ✅ Web app deployed screen with a URL that looks like:

https://script.google.com/macros/s/XXXXXXXXXXXXXXX/exec

Copy that URL — you'll need it for the next step.

6Add the URL to the Form

Open info_request_form.html in a code editor and find this line near the bottom:

const GOOGLE_SCRIPT_URL = 'YOUR_GOOGLE_SCRIPT_URL';

Replace YOUR_GOOGLE_SCRIPT_URL with your actual web app URL, for example:

const GOOGLE_SCRIPT_URL = 'https://script.google.com/macros/s/XXXXXXXXXXXXXXX/exec';
✅ That's it! Form submissions will now be sent directly to john@orangeagency.media.

7Test It

Open the form in your browser (you can open it directly from the HTML file). Fill it out and submit — you should receive an email within seconds.

If you don't want to wait for a real submission, click the Run testEmail button in the Apps Script editor (the play ▶ button) to send a test message immediately.

🔧 Troubleshooting

Not receiving emails?

  • Make sure you set "Who has access" to Anyone during deployment
  • Check your spam/junk folder
  • Run the testEmail() function from the editor to verify sending works
  • Re-deploy after making code changes (create a New deployment to get a new URL)

Need to update the recipient email?

Edit the RECIPIENT_EMAIL constant at the top of the Apps Script and create a new deployment.
